MY GARAGE

Smart Garage Booking & AI Filtering System

View Demo

Abstract

MY GARAGE is a smart web-based platform developed to help users easily find garages, book services, and manage vehicle maintenance. The system also includes an AI assistant to guide users to the appropriate garage services.

Key Features

Browse Garages
Booking System
AI Assistant
Admin Dashboard
Garage Owner Dashboard
Real-time Notifications
